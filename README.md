# 📅 Day 1 – Python Basics

## 🐍 What is Python?

Python is a **high-level, interpreted programming language** created by Guido van Rossum. It is widely used because it is simple, readable, and powerful.

## 📌 Why is it called Python?

Python is not named after the snake 🐍. It is named after a comedy show called **“Monty Python’s Flying Circus”**.

---

## 💡 What is a Programming Language?

A programming language is a way to communicate with a computer to give instructions.

---

## ⚙️ Types of Programming Languages

### 1. Compiler Language

* Whole program is converted into machine code at once
* Example: C, C++

### 2. Interpreter Language

* Code is executed line by line
* Example: Python, JavaScript

---

## 🧠 Basic Concepts Learned

### 📦 Variables

Variables are containers used to store data.

```python
name = "Ali"
age = 20
```

---

### 📊 Data Types

* int → numbers (10, 20)
* float → decimal numbers (2.5, 3.14)
* str → text ("Hello")
* bool → True/False

---

### ⌨️ Input / Output

```python
name = input("Enter your name: ")
print("Hello", name)
```

---

### 🔤 Strings

Strings are text inside quotes.

```python
text = "Hello World"
```

---

### ✂️ String Indexing

Each character has an index starting from 0.

```python
text = "Python"
# P = 0, y = 1, t = 2
```

---

### ✂️ String Slicing

Used to get a part of a string.

```python
text = "Python"
print(text[0:4])  # Pyth
```

# 📅 Day 2 – Operators in Python

## ➕ Arithmetic Operators

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Modulus (%)
* Floor Division (//)
* Exponentiation (**)

---

## 🧾 Assignment Operators

Used to assign values to variables.

### Example:

```python
x = 5
x += 2
```

### Operators:

* +=
* -=
* *=
* /=
* //=
* %=
* **=

---

## ⚖️ Comparison Operators

Used to compare values and return True/False.

* ==
* !=
* >
* <
* > =
* <=

---

## 🔗 Logical Operators

Used to combine conditions.

* and → both true
* or → one true
* not → reverse result

---

## 🔀 Conditional Statements

Used for decision making.

```python
if condition:
    print("True")
else:
    print("False")
```
