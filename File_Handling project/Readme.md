# 📁 Python File Handling Management System

A beginner-friendly Python project that demonstrates complete **File Handling Operations** using Python and the `pathlib` module.

This project allows users to perform CRUD operations on files directly from the terminal/command line.

---

# 🚀 Features

✅ Create Files
✅ Read File Content
✅ Update Existing Files
✅ Rename Files
✅ Append Data to Files
✅ Delete Files
✅ Display All Files & Folders
✅ Exception Handling Support

---

# 🛠️ Technologies Used

* Python 3
* pathlib Module
* os Module

# ▶️ How to Run

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/file-handling-system.git
```

## 2️⃣ Open Project Folder

```bash
cd file-handling-system
```

## 3️⃣ Run Python File

```bash
python main.py
```
