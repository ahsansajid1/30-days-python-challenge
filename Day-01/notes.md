# 📘 Day 1 – Python Basics Notes

## 🐍 Introduction to Python

Python is a high-level, interpreted programming language created by **Guido van Rossum**. It is easy to learn, simple, and widely used in web development, data science, AI, and automation.

---

## 📌 Why Python is called Python?

Python is named after the comedy show **"Monty Python’s Flying Circus"**, not the snake.

---

## 💡 What is a Programming Language?

A programming language is a set of instructions used to communicate with a computer.

---

## ⚙️ Types of Programming Languages

### 1. Compiled Language

* Code is converted into machine language at once
* Faster execution
* Example: C, C++

### 2. Interpreted Language

* Code runs line by line
* Easier to debug
* Example: Python, JavaScript

---

## 🧠 Basics Learned

### 📦 Variables

Variables are containers used to store data.

```python
name = "Ali"
age = 20
```

---

### 📊 Data Types

* int → whole numbers (10, 20)
* float → decimal numbers (2.5, 3.14)
* str → text ("Hello")
* bool → True/False

---

### ⌨️ Input / Output

```python
name = input("Enter your name: ")
print("Hello", name)
```

---

### 🔤 Strings

Strings are text inside quotes.

```python
text = "Hello World"
```

---

### 🔢 String Indexing

Each character has an index starting from 0.

```python
text = "Python"
# P = 0, y = 1, t = 2
```

---

### ✂️ String Slicing

Used to extract part of a string.

```python
text = "Python"
print(text[0:4])  # Pyth
```
